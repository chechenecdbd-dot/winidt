# winidt (Main app for GTA5RP players capt/biz)

Скрытый триггер-бот на C++, реагирующий на изменение цвета прицела в центре экрана.  
Использует **Desktop Duplication API (DXGI)** для захвата пикселя с минимальной нагрузкой на систему.  
Позволяет запомнить два цвета прицела (нейтральный и вражеский) с помощью одной кнопки, поддерживает переназначение клавиши-триггера и не оставляет следов в системе.

##  Особенности

- **Мгновенный захват пикселя** – через DXGI Duplication (1×1 пиксель с GPU), без полных скриншотов.
- **Циклическое сохранение цветов** – одно нажатие сохраняет нейтральный цвет, следующее – цвет врага.
- **Переназначаемый триггер** – можно выбрать любую клавишу/кнопку мыши для активации.
- **Скрытая работа** – нет окон, нет консоли, нет лог-файлов.
- **Расширенная автоочистка следов** – удаляет временные файлы, Prefetch‑файлы и записи в реестре (включая `AppCompatFlags`, `MuiCache`, `ComDlg32\OpenSavePidlMRU`, `ComDlg32\LastVisitedPidlMRU`).
- **Маскировка** – (опционально) подмена имени процесса и запрет на чтение памяти другими процессами.
- **Совместимость** – Windows 10/11 (все сборки, 1507–25H2 и новее).

##  Горячие клавиши

| Клавиша          | Действие |
|------------------|----------|
| `Delete`         | Включить / выключить бота (звуковой сигнал: 1000 Гц – вкл, 500 Гц – выкл) |
| `F1`             | Сохранить цвет прицела (первое нажатие – нейтральный, второе – вражеский) |
| `F2`             | Переназначить клавишу-триггер (нажать, затем нужную кнопку/клавишу) |
| `Ctrl+Shift+Q`   | Завершить программу |

Триггер по умолчанию: **правая кнопка мыши (ПКМ)**.  
Звуковые сигналы подтверждают все операции.

##  Как использовать

1. **Запустите `windinit.exe`** (программа работает в фоне, без окон).
2. **Запомните цвета прицела:**
   - Наведите прицел на нейтральную область (стена, небо), где цвет не меняется.
   - Нажмите `F1` — услышите низкий звук (800 Гц). Сохранён нейтральный цвет.
   - Наведите прицел на врага (цвет должен измениться).
   - Снова нажмите `F1` — услышите высокий звук (1200 Гц). Сохранён вражеский цвет.
   - Повторные нажатия `F1` будут циклично перезаписывать нейтральный → вражеский → нейтральный...
3. **Активируйте бота:** нажмите `Delete` (звуковой сигнал включения).
4. **В игре зажмите настроенную клавишу-триггер** (по умолчанию ПКМ).  
   Когда цвет в центре экрана совпадёт с сохранённым вражеским цветом, бот автоматически нажмёт левую кнопку мыши (ЛКМ).
5. **Переназначение триггера (опционально):**
   - Нажмите `F2` (два коротких сигнала).
   - Нажмите любую клавишу клавиатуры или кнопку мыши.
   - Успешное назначение подтверждается высоким сигналом (1500 Гц). Если в течение 5 секунд ничего не нажато – низкий сигнал ошибки.
6. **Выход:** `Ctrl+Shift+Q`.
